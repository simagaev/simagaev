##  Ключевые проекты

###  Автоматизированная система сбора и визуализации данных для финансовой отчётности
**Задача:** Консолидировать данные из банков, CRM и Excel в единое хранилище.  
**Решение:** Спроектировал схему данных, реализовал ETL с инкрементальной загрузкой (SQLAlchemy), настроил партиционирование в PostgreSQL.  
**Результат:** Время отчётов — с **10 дней → 2 часа**, система держит **500K+ записей**.  
**Стек:** `Python` `SQLAlchemy` `PostgreSQL` `Redis` `Grafana`

---

### Бэкенд для социальной платформы «Yatube»
**Задача:** API для блогов с постами, комментариями, подписками.  
**Решение:** REST API на Django REST Framework с JWT, пагинацией, фильтрацией, тестами (покрытие >80%).  
**Результат:** Выдерживает **1000 запросов/мин**, задокументирован в Swagger.  
**Стек:** `Django` `DRF` `PostgreSQL` `Docker` `Nginx` `GitHub Actions`

---

### Парсер PEP (scrapy_parser_pep)
**Задача:** Собирать и анализировать статусы Python Enhancement Proposals.  
**Решение:** Паук на Scrapy с обработкой пагинации, выгрузка в CSV и PostgreSQL.  
**Результат:** Собрал статистику по **500+ PEP**, автоматизировал еженедельный мониторинг.  
**Стек:** `Scrapy` `BeautifulSoup4` `PostgreSQL` `Docker` `GitHub Actions`

---

### Foodgram — сервис рецептов (командный проект)
**Задача:** Платформа для публикации рецептов, подписок и скачивания списка покупок.  
**Решение:** Бэкенд на Django + DRF, Redis для кеширования, генерация PDF.  
**Результат:** Проект в тестовой эксплуатации с **500+ пользователями**.  
**Стек:** `Django` `DRF` `PostgreSQL` `Redis` `Docker` `Nginx` `GitHub Actions`

### QRkot-spreadsheets — автоматизация документооборота
